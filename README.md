# 75 Hard Fitness Analytics

## Overview

A personal data analytics project tracking my 75 Hard challenge progress.

This project uses data engineering and analytics tools to collect, store, process and visualise my fitness journey.

## Objectives

Track and analyse:

- Daily 75 Hard habit completion (Starting 20th/07/2026)
- Gym progression (Upper/Lower/Push/Pull)
- Running performance
- Body composition changes
- Training consistency
